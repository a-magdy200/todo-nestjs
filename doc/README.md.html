Description

Basic TODO App built with Ionic (React), Nestjs, TypeORM

Installation 

$ yarn install

Environment

Create a local database, and create an .env file similar to .env.example Example:

DATABASE_HOST=localhost
DATABASE_PORT=3306
DATABASE_USER=juleb_user
DATABASE_PASS=juleb
DATABASE_NAME=julebdb

Running the app 

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
